Customer Churn Prediction Using ML

This project focuses on predicting customer churn using machine learning algorithms such as Decision Trees, Random Forest, and AdaBoost. It aims to identify patterns in customer behavior that indicate a likelihood of leaving a service.

By analyzing historical data, the model helps organizations retain customers by addressing issues and enhancing service delivery.

This project employs Python for model training, along with SQL for data handling, and uses various data visualization tools to present insights.
